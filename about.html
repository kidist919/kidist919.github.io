Project Inspiration

Security issue is the basic problem in most of the company. To solve such issues, the task management system is developed to comunicate the manager and worker secretly. The assigned task is known only between the worker and manager. Once the worker finish its work, he/she submit its report using the system. Hence, such system create an interesting working environment in a given company.
